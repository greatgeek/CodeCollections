# HTTP 响应代码

HTTP 响应状态代码指示特定 [HTTP](https://developer.mozilla.org/zh-cn/HTTP) 请求是否已成功完成。响应分为五类：

**信息响应**(`100`–`199`)，

**成功响应**(`200`–`299`)，

**重定向**(`300`–`399`)，

**客户端错误**(`400`–`499`)和

**服务器错误** (`500`–`599`)。

状态代码由 [section 10 of RFC 2616](https://tools.ietf.org/html/rfc2616#section-10)定义。
