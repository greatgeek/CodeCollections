# 滑动窗口的最大值

给定一个数组 nums 和滑动窗口的大小 k，请找出所有滑动窗口里的最大值。

示例:
```
输入: nums = [1,3,-1,-3,5,3,6,7], 和 k = 3
输出: [3,3,5,5,6,7] 
解释: 

  滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

提示：

你可以假设 k 总是有效的，在输入数组不为空的情况下，1 ≤ k ≤ 输入数组的大小。

**Show me the code**

```java
    public int[] maxSlidingWindow(int[] nums, int k) {
        if(nums.length==0 || k==0) return new int[0];
        Deque<Integer> deque = new LinkedList<>();
        
        int[] res = new int[nums.length-k+1];
        int index=0;
        for(int i=0;i<k;++i){ // 形成第一个窗口
            while(deque.size()>0 && nums[i]>nums[deque.peekLast()]) deque.removeLast();
            deque.addLast(i);
        }
        res[index++]=nums[deque.peekFirst()];

        for(int i=k;i<nums.length; ++i){
            if(i-deque.peekFirst()>=k) deque.removeFirst();
            while(deque.size()>0 && nums[i]>nums[deque.peekLast()]) deque.removeLast();
            deque.addLast(i);
            res[index++]=nums[deque.peekFirst()];
        }
        return res;
    }
```
