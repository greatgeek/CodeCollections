### 对一个字符串产生回文判断

```java
public boolean[][] generatePalindrome(String ss){
    char[] s = ss.toCharArray();
    int n = s.length;
    if(n==0){
        return 0;
    }
    
    boolean[][] isPalin = new boolean[n][n];
    int i,j,t;
    for(i=0;i<n;i++){
        for(j=i;j<n;j++){
            isPalin[i][j]=false;
        }
    }

    // generate palindrome
    for(t=0;t<n;t++){
        // odd-length
        i=j=t;
        while(i >= 0 && j < n && s[i] == s[j]){
            isPalin[i][j]=true;
            i--;
            j++;
        }

        // even-length
        i=t;
        j=t+1;
        while(i>=0 && j<n && s[i]==s[j]){
            isPalin[i][j]=true;
            i--;
            j++;
        }
    }
    return isPalin;
}

```
